# ownAI_assessment for nodejs(backend)
first check the node -v => v20.19.4
run the following command to run the project 
npm i
npm init -y
npm install express
npm install sequelize
npm install mySql2
npm install bcryptjs jsonwebtoken
Then run db.js => node db.js
